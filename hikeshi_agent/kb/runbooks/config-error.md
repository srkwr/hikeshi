# Runbook: 設定・環境変数の誤り（config error）

（合成サンプル KB・実データではない）

## 症状
- 特定機能の失敗・タイムアウト・接続エラー。直近の config_change（タイムアウト値・レプリカ数・
  SMTP 設定・接続文字列・機能フラグ等）に相関。コードのデプロイ自体は問題ない。

## 根本原因
- 設定・環境変数の誤り。category = config_error。

## 推奨対処
- **設定修正（config_fix）**：誤った設定値を正しい値へ。要 HITL。

## 注意
- コードのロールバックでは直らない（設定が原因のため）。直近の設定変更履歴を確認する。
